# AzureServiceHealthAlertsBicepCreator

Simple website to visually create bicep file for setting up service health alerts in Azure

## Testing bicep file

Once you copy the output to bicep file (use .bicep extension!) you can test in vsc using [bicep extension](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-bicep) or using Azure CLI using [az bicep](https://learn.microsoft.com/en-us/cli/azure/bicep?view=azure-cli-latest)
